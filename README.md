# Land-Cover Map Intercomparison (LC Intercomparison)

This project is a **Python library** for comparing land-cover maps, specifically **ESA CCI** and **GLC_FCS30** datasets.  
It computes:

- Confusion matrices
- Accuracy metrics:
  - Overall Accuracy (OA)
  - Kappa coefficient
  - Producer’s Accuracy
  - User’s Accuracy
